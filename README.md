# Student Classification - Machine Learning Project

This project predicts whether a student will **pass or fail** based on their math, reading, and writing scores.

## Features
- Data cleaning and exploratory analysis (EDA)
- Visualization of score distributions and correlations
- Classification using **Random Forest Classifier**
- Evaluation with accuracy, confusion matrix, and classification report
- Built and tested in **Google Colab**
